Welcome to my GitHub Pages

The main project i'm working on is Gopei (Golang Programming Environment Installer) for Ubuntu but also at a few other projects in golang.

To use Gopei and start programming in go with full git support download installer from here and unarchive.

The main command to install golang and ide suite is:

$ ./Install.sh -g

No root rights are required.

Remember to read the User Manual to see all the futures of the installer.
